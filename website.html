The common markup language used to build and design web pages is called HTML, or HyperText Markup Language. Using a system of tags or components to format text, pictures, video, links, and other elements inside a web document, it gives material on the internet a structure. That is why, all of the course members have completed 100% of CodeAcademy HTML course with success as the first step of this Website creating project. Thus, we have used HTML to determine the primary structure of the pages of our website and we have provided and formated all of the text, pictures, video, hyperlinks, and other elements inside this website with the help of HTML.

Cascading Style Sheets, or CSS for short, are a language for style sheets that specify how an HTML or XML document (including XML dialects like SVG or XHTML) is presented. It offers a method for managing the look, style, and organization of web pages. That is why, all of the course members have completed 100% of CodeAcademy CSS course with success as the second step of this Website creating project. Thus, we have used CSS to manage the look, style, and organization of the pages of our website.

The primary use of JavaScript, a dynamic, high-level programming language, is the creation of interactive and dynamic online content. It is a fundamental technology in web development, much as markup language HTML and style sheet language CSS. That is why, all of the course members have completed 100% of CodeAcademy JavaScript course with success as the third step of this Website creating project. Thus, we have used JavaScript to create all of the interactive and dynamic online content in all of the pages of our website.
